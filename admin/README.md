# Liberty Admin Panel

Админ-панель для управления сайтом Liberty.

## Установка

```bash
cd admin
npm install
```

## Запуск

```bash
npm start
```

Приложение откроется на http://localhost:3000

## Структура

- `src/Components/` - Компоненты (Layout, Sidebar, Header)
- `src/Pages/` - Страницы админки (Dashboard, Products, Orders, Users, Settings)
- `src/Routes/` - Роутинг
- `src/Redux/` - Управление состоянием (если нужно)

## Функционал

- Главная страница с статистикой
- Управление товарами
- Управление заказами
- Управление пользователями
- Настройки сайта
