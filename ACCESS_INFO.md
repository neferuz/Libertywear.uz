## Как поделиться доступом

1. **Для клиентской части (магазин):**
   - Отправьте ссылку: `http://147.45.155.163:3000`
   - Любой человек сможет открыть эту ссылку в браузере

2. **Для админ панели:**
   - Отправьте ссылку: `http://147.45.155.163:3001`
   - Требуется авторизация

3. **API документация:**
   - Swagger UI: `http://147.45.155.163:8000/docs`
   - ReDoc: `http://147.45.155.163:8000/redoc`

## Важно!

- Убедитесь, что порты 3000, 3001 и 8000 открыты в firewall
- Для постоянного доступа рекомендуется настроить домен и SSL сертификат
- В продакшене рекомендуется использовать nginx как reverse proxy

## Запуск сервисов

### Backend:
```bash
cd backend
python main.py
```

### Frontend (Zara):
```bash
cd zara
npm start
```

### Admin:
```bash
cd admin
npm start
```
